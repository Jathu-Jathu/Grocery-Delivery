# 🛒 Grocery Delivery Web Application 🚀

A full-stack grocery delivery platform built with modern web technologies.  
This project provides a complete online grocery shopping experience with user, admin, and delivery partner functionalities.

🌐 Live Demo: 

---

## 📌 Project Overview

This project is a real-world grocery delivery system where users can browse products, add items to cart, place orders, make payments, and track deliveries.

The system includes:

👤 User Panel  
🛠️ Admin Dashboard  
🚴 Delivery Partner Dashboard  
📦 Order Management System  
📍 Live Order Tracking  

---

## ✨ Features

### 👤 User Features

✅ User Registration & Login  
✅ Secure Authentication  
✅ Browse Grocery Products  
✅ Product Search  
✅ Product Details View  
✅ Add To Cart  
✅ Update Cart Quantity  
✅ Manage Addresses  
✅ Checkout System  
✅ Stripe Online Payment  
✅ View Order History  
✅ Live Order Tracking  

---

### 🛠️ Admin Dashboard

✅ Admin Authentication  
✅ Manage Products  
✅ Add / Update / Delete Products  
✅ Manage Orders  
✅ Manage Users  
✅ Manage Delivery Partners  
✅ View Sales Information  

---

### 🚴 Delivery Partner Dashboard

✅ Delivery Partner Login  
✅ View Assigned Orders  
✅ Update Delivery Status  
✅ Share Live Location  
✅ Track Delivery Progress
